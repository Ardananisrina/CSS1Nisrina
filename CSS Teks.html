Font Style

CSS dapat mengatur perataan teks yang ada pada halaman web yang dibuat. Tetrdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.

  • text-align: digunakan untuk mengatur perataan teks, vallue yang dapat digunakan yaitu left, righht, center.
  • color: digunakan untuk memberikan warna pada teks.
  • letter-spacing: mengatur jarak antar karakter.
  • word-spacing: mengatur jarak antar kata.
